# Preview Project Workflow

This workflow details the exact commands needed to start the preview dev server and test "The Scriptorium" Next.js project.

## Steps

1. **Prerequisites**
   Ensure all dependencies are installed and the project is ready:
   ```bash
   npm install
   ```

2. **Start the Development Server**
   Run the development server on port 9002:
   ```bash
   npm run dev
   ```

3. **Verify the App**
   Open your browser and navigate to:
   http://localhost:9002
